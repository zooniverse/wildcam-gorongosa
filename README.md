# WildCam Gorongosa

Built using:

- Gulp
- Webpack
- Coffeescript, CJSX
- React.js
- Reflux
- Stylus

---

## Installation

Clone and run `npm install`. Use `npm start` to run locally.

## Development

Oauth will only work if you use `local.zooniverse.org`, so you will need to remap localhost in your `/etc/hosts` file.
